


# Crop to Fertilizer Recommendation App

This app recommends fertilizers for crops based on nutrient requirements and environmental factors.

## Folder Structure
- `data/`: Contains the input datasets (`Crop_recommendation.csv`, `Fertilizer Prediction.csv`).
- `models/`: Placeholder for saved models (if needed in the future).
- `notebooks/`: Jupyter notebooks for training or analysis (optional).
- `main.py`: Streamlit app for fertilizer recommendations.
- `requirements.txt`: List of dependencies.
- `README.md`: Project documentation.
